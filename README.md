### About

Bat Outta Hell is an audio-first arcade driving game made to be accessible to screen reader users including those who are blind or have visual impairments. It supports play with the keyboard, and a controller. Haptics are available if using a supported controller.

### AI Usage Disclosure

Significant portions of this game have been coded using an LLM.

### Play the Game

You can play Bat Outta Hell by clicking [here](https://ironcross32.github.io/Bat_Outta_Hell/) Instructions are available by clicking, "How to play" on that page.

Playing on mobile is in early days and is very much experimental; however, playing on Android is possible with an Xbox controller or similar at this time. Turning VoiceOver off is necessary on iOS to play with touch controls. To this end, there's a tts option which will force the game to output through the system text-to-speech voice.

### Graphics

This game contains rudimentary graphics and should be playable by those with sight. However, owing to the fact the developer is blind, any issues with the visuals should be reported in an issue.
